Portfolio

Here are some of the projects I’ve worked on:

  • Project 1: Hospital Appointment Management System
     In response to the challenges faced by hospitals during the COVID-19 pandemic, the team developed a hospital management system focused on emergency appointment placement. Users can easily book appointments via a simple form, with confirmations provided instantly. The system uses Flask for the back-end, HTML for the front-end, and MySQL for secure data storage. It features a user-friendly interface for patients and an admin portal for managing records.
  • Project 2: WeatherApp
     WeatherApp enhances the user experience by introducing dynamic weather icons and wind speed integration, addressing limitations in existing weather apps. It provides real-time, location-based weather updates with intuitive design, offering a more comprehensive view of current and predicted conditions.
  • Project 3: Guessing Game
     The Number Guessing Game is a Python-based application where players guess a randomly selected number between 1 and 20. Built with Python and Tkinter, it provides interactive feedback on guesses and runs easily with basic setup instructions.
